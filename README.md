# Vaulted
**Real-Time Market Value Analysis for Your Treasures**

Vaulted is a collector-driven, AI-assisted powerhouse for collectors of various fandoms to harness control of their amassed collections in one simple, easy to use, intuitive platform. AI pulls real-time pricing data for any specific collectible within a supported category to give a clear, concise, and calculated value within minutes.

Currently, the platform is designed to support the following collectible categories: Hot Wheels, Pokemon cards, and sneakers.
